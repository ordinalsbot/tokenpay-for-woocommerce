== OrdinalsBot for WooCommerce ==

This plugin allows stores that use Wordpress WooCommerce shopping cart system to accept Bitcoin and runes via OrdinalsBot.

= Features =

* Payment amount is calculated using real-time exchange rates.
* Supports on-chain runes and bitcoin transactions

*In order to use this plugin you have to create an account and API Key for tokenpay. Reach out to us at https://discord.ordinalsbot.com so we can share your tokenpay API Key with you.*

== Installation ==

1. Install the WooCommerce OrdinalsBot plugin:
  * [Upload ZIP](https://codex.wordpress.org/Plugins_Add_New_Screen#Upload_Tab): Admin » Plugins » Add New » Upload Plugin
  * Extract ZIP and upload extracted directory to the `/wp-content/plugins/` through FTP
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enter [API Credentials]() to WooCommerce-OrdinalsBot Plugin Settings: Admin » WooCommerce » Click on Checkout tab » Find "OrdinalsBot" in Payment Gateways table » Click "Settings"
6. Check "Enable Cryptocurrency payments via OrdinalsBot" checkbox.
7. Make sure to be using woocommerce NOT in "block editor mode", otherwise the option will not show up at checkout. You can change this on the woocommerce > checkout page settings.
